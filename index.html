This was a quick project by Kevin Greene, inspired by a Strangeloop talk by Sarah Groff-Palermo.

In this example I pulled the HipChat logs of Spantree and processed how often people talk. Each time a bubble lights up, it represents someone sending a chat message. My main goal was to get a small insight into conversations and flow of messages. Each bubble represents a different person, except for the lower right, which is all people not actively employed by Spantree (clients / friends).

Time is calculated logarithmically, so there aren't long boring periods. Currently the entire thing takes about 1000 seconds, so strap in, or leave it up on another monitor if you are interested in seeing it to the finish.
